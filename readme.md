Entitys, Components, Systems
============================

Basic test for an Entity Component System


Running
===========

Run with live server

```
npm install -g live-server
live-server
```

Open new Tab

```
gulp watch
```


Testing
=======

```
npm install -g karma
gulp test
```

Running one test in isolation

```coffeescript
fdescribe '...', ->
    ffit '...', ->
```
